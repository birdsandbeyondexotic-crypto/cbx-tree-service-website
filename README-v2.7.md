# CBX Tree Service Website v2.7

## Phase 1–4 foundation release

### Conversion improvements
- Three-button mobile action dock: Text, Free Estimate and Call
- Dismissible emergency-service banner
- Text-message links in the header, navigation and footer
- Stronger mobile estimate flow

### Trust and customer education
- Interactive FAQ section
- Real-project spotlight using CBX photography
- Expanded emergency, cleanup, insurance and service-area answers

### Local SEO foundation
- Homepage title, meta description and canonical URL management
- Search-focused homepage copy retained for tree trimming, removal and Southwest Florida service areas
- Existing service and community landing pages remain connected

### Visual and mobile redesign
- Compact mobile header and navigation drawer
- Mobile hero image and content reflowed so the headline remains readable
- Swipeable service, trust, process and statistics cards
- Cleaner mobile footer and fixed conversion dock
- Subtle scroll-based entrance animation with reduced-motion support

## Deployment
Upload the changed files to the root of the GitHub repository and commit to `main`. The existing GitHub Actions workflow will build and deploy the site to Firebase Hosting.
