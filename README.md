# CBX Tree Service Website Prototype

This folder contains a responsive one-page website prototype built from CBX's supplied logo, company information and jobsite photos.

## Files
- `index.html` — page content and structure
- `styles.css` — full responsive design
- `script.js` — mobile navigation and estimate-request email fallback
- `assets/` — logo, owner photo and jobsite photos

## Preview
Open `index.html` in a modern browser.

## Before launch
1. Replace the email-based estimate form with the official Jobber request form or booking link.
2. Confirm the preferred primary phone number.
3. Add verified Google reviews.
4. Add official license/insurance wording and any license number that should be displayed.
5. Compress images to WebP for production speed.
6. Connect analytics, Google Search Console and the production domain.
